# sfx/ (local only)

Royalty-free sound effects and recorded voice drops live here — shared
across all shows. Audio files are gitignored; this README logs every clip's
source URL and license so mixes can be rebuilt after a fresh clone. Also
list each clip in `../library.md` under the show(s) that use it.

## Good free sources

- Pixabay (royalty-free, no attribution): https://pixabay.com/sound-effects/
- Orange Free Sounds: https://orangefreesounds.com/
- Voice drops ("Let's Get Ready to Rumble"-style announcer lines) are rarely
  legit free downloads — record your own.
