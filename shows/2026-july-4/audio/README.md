# audio/ (local only)

**Only the final show mix belongs here** (plus its `*.analysis.json` timing
files, which are the one thing committed). Everything reusable — purchased
song MP3s, SFX, voice drops — lives in the shared library instead:
`music/tracks/` and `music/sfx/`.

Rebuild instructions live in the show's music plan; track list in
`music/library.md`.
