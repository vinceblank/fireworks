## Show-night ritual (3 presses: wake · strobe · show)

1. Key ON → TEST: continuity **all green** (red LED = unwired scripted cue — fix before arming)
2. **ARM** (wait for solid LED + device count 2)
3. **PRESS STEP (wake)** → nothing fires; display queues the strobe row
4. **PRESS STEP again** → strobe lights (show-live beacon)
5. Start the mix on the phone → BT speaker (volume pre-set)
6. **PRESS STEP at mix 0:06** — 2 s before the first lyric; hearing the lyric right after = you nailed it. The NBB wall erupts and the clock runs the rest.
7. Hands off. Drift → **+/−** buttons. Do NOT press STEP again (it skips ahead). DISARM = abort.

## No-wires dry-run (bench test — do this before setup day)

**Knob state**: Station 1 key → **ARM**, dial **CH 01** · Station 2 key →
**ARM**, dial **CH 02** · 18R2 key ON, channel **CH 01** (the trigger
channel), all terminals EMPTY. (ARM is safe with nothing wired — fired cues
are 0.1 s pulses into empty terminals. TEST-keyed modules won't run scripts.)

1. Each module: firing mode = e-match `0.1` (hold SYNC+CH− to set)
2. 18R2 TEST: expect **every scripted cue RED** (scripted, nothing wired —
   correct). Count: 15 red on CH01, 14 on CH02 = full cue coverage
3. 18R2 **ARM** → ARM LED blinks (script downloading to modules) → wait for
   **solid ARM + device count 2**
4. Run the real ritual: **STEP (wake — nothing fires)** → **STEP** (CH01-16
   LED flips red = strobe) → start the mix → **STEP at mix 0:06** → hands
   off 10:53; follow events on the remote display against the music
5. After: all 29 cue LEDs fired-red on the modules = every command delivered
6. **Second pass — failure practice**: mid-show TEST/DISARM (abort) →
   re-ARM → restart → **STEP** to catch up to the music; try one **+/−** nudge

## Checklists

**Wiring-complete walk** — every row above has: product staked/braced, e-match seated, wire to the right cue terminal, fan aim checked. Tick each row.

**Continuity (local)** — each 18M: power on, verify all wired cues show continuity at the module.

**Continuity (remote)** — 18R2 TEST mode: every scripted cue green. Any red = find it now, not at ARM.

**Audio dry-run (afternoon, DISARMED)** — press-timing practice only (disarmed modules don't run the script): play the mix and rehearse the 0:06 press by ear a few times. The full-system rehearsal is the no-wires bench dry-run above.

**Arming (dusk)** — perimeter clear → strobe area clear → key ON → TEST green → ARM → ritual above. Site is HOT from ARM to show end + 15 min. Misfires: nobody approaches for 15 minutes, then water.
